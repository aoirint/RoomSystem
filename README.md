## 試験開発システム（2020/11、開発終了）
ESP32とFirebaseを使ったシステムを構築していました。
試験の結果、プログラム更新するにはハードウェアに接続する必要が出てくるため、
シリアル通信でいいじゃないか、ということになりました。
センサをたくさん付けるような場合には物理接続の手間がないこちらの方がよさそうです。
